# Sonawane Power System - Solar Quotation Generator

A professional React.js application for generating solar rooftop quotations.

## Features
- Real-time calculations for solar system pricing and subsidy.
- Dynamic Bill of Materials (BOM) management.
- Daily/Monthly unit generation estimates.
- Professional PDF generation with customer details and dynamic filenames.
- Responsive design with a premium UI.

## Tech Stack
- React.js
- Vite
- Vanilla CSS

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
